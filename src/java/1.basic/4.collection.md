# JAVA 基础 - 集合

## 1.迭代器Iterator
迭代器是在Collection接口中定义的，所有实现了Collection的类都具有此功能（List、Set），迭代器的主要功能就是单向轮训Collection中所有的元素。
主要用处就是在边轮训的时候边移除其中的元素。
```java
Iterator<String> it = testList.iterator();
while(it.hasNext()){
    String temp = it.next();
    if (temp.equals("a") || temp.equals("d")){
        it.remove();
    }
}
```
